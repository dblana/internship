Forecasting

In this document, we will present forecasting methods that can be used with time series data. Here, we will illustrate how the algorithm is working by forecasting evolution of hospitalizations due to COVID-19, in the state of Colorado. To do so, we are using data available on covidtracking.com. This document is written to be adapted to your own dataset and forecasting needs.

Here, you can define a dependent variable dep_var, that will be forecast with respect to an explanatory variable exp_var.

dep_var <- 'hospitalizedCurrently'
exp_var <- 'date'

Now, you have to convert data to make it usable by our models. You can also filter data. Here we use it to get only data related to Colorado. If you don’t need to do so, you can just write data <- as_tsibble(data).

data <- as_tsibble(data %>% filter(state=="CO"), index = date)
data[dep_var] <- na.interp(data[dep_var])
data %>% ggplot(aes_string(x = exp_var, y = dep_var))+
  geom_line()+  ggtitle('Evolution of hospitalization')+
  xlab('Date')+

The method we will be using to forecast is the exponential smoothing. This general method however presents some variations. We will first test a few of them, before choosing the one that might give us eh best results.

data %>%
  stretch_tsibble(.init = 10) %>%
  model(
    SES = ETS(hospitalizedCurrently ~ error("A") + trend("N") + season("N")),
    Holt = ETS(hospitalizedCurrently ~ error("A") + trend("A") + season("N")),
    Damped = ETS(hospitalizedCurrently ~ error("A") + trend("Ad") +
                   season("N"))
  ) %>%
  forecast(h = 1) %>%
  accuracy(data)

To chose our method, we need to observe the MSE and the MAPE computed for each of them. I our case, Holt and Damped seem to produce similar accuracies. We decide here to keep the Damped one. We finally can forecast data, after we decided how many values we would like to predict. This values is held in the frc_length variable. The following code can now plot the forecast evolution.

frc_length <- 30

frc <- data %>% 
  model(Damped = ETS(hospitalizedCurrently ~ error("A") + trend("Ad") + season("N"))) %>% 
  forecast(h = frc_length)

frc %>%
  autoplot(data) +
  labs(x="Date", y="Number currently hospitalized people",
       title = "Evolution of currently hospitalized people")
